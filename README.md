# Kred

Kred is the Internet of Credit - a protocol enabling instant, programmable capital deployment through KUSD, a stablecoin that earns yield from real-world financial activity.

## Getting Started

### Install Dependencies

```shell
$ forge install
```

### Build

```shell
$ forge build
```

### Clean

```shell
$ forge clean
```

### Check Formatting

```shell
$ forge fmt --check
```

### Format

```shell
$ forge fmt
```

### Anvil

```shell
$ anvil
```

### Cast

```shell
$ cast <subcommand>
```

### Help

```shell
$ forge --help
$ anvil --help
$ cast --help
```
